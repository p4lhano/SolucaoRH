# SolucaoRH

Sistema de Gestão de RH, sistema desenvolvido com requisito para obtenção de notas para disciplina Desenvolvimento de Software Visual.
___
### Para executar o Back-end
Entre em API
#### Execute :
#dotnet restore
#### #dotnet what run
___
### Para executar o Front-end
Entre em Front
#### Execute :
#npm install
#### #ng serve --o
